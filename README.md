# auto-table-form

## Describe

### 简述

此项目将表单的生成过程可视化，使得一份表单的生成过程更简便，且可以直接应用在具体的项目中。 可以在页面中配置各项表单项并生成相应的子组件和配置文件，子组件和配置文件放到表单项中以后可以自由使用。

### 项目发展

#### 2020.04.29

##### 依赖项

表单 UI：element-ui 子组件格式：vue 配置文件：javascript

##### 功能

element-ui 按需引入
表单项：输入框(文字、数字)、下拉选项框
